Just Random

Random evening planner, with three integrated APIs, one for each section. The responsive website, created with react, finds a film, a meal, and a drink for the user, depending on their selected categories. (catergories for film yet to integrated)

React router and the useSearchParams hook is used, to work with search params in the url.

wordle-six

Rebuild of the wordle, a puzzle game from The New York Times, that makes use of react with multiple react hooks and jsx.

Utilising local-storage to store the selected theme and an array of words, the website is responsive with a hamburger menu (mobile view) allowing the player to change the theme of the website.

A dictionary-API is also used to check if the given word exists.
